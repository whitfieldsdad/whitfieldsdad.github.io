Translating STIX 2 content into directed acyclic graphs (DAGs)

STIX 2 models relationships between STIX Domain Objects (SDOs) and STIX Cyber-observable Objects (SCOs) in two different ways: Using STIX Relationship Objects (SROs) (i.e. within relationship or sighting objects); and Using embedded relationships (e.g. attack-pattern objects contain embedded references to x-mitre-tactic objects using the kill_chain_phases property). In this post, we’ll cover: The set of SDOs, SROs, SCOs, and predicates outlined within the STIX 2.1 specification; The set of SDOs, SROs, and predicates used by MITRE when expressing ATT&CK (Enterprise, Mobile, ICS), CAPEC, NIST SP 800-53, and the relationships between ATT&CK techniques and NIST SP-800-53 controls in STIX 2.
